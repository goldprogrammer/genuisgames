Checklist for Securing Your Windows PC

Set passwords

Make a secure password
  • At least three of the following:
    • numbers
    • capital letters
    • lower letters
    • symbols
    • symbols in the middle
  • To better remember your password, use a passphrase such as "where's the beef?" This could become "Wh D@ b33f?" this would let you remember the passphrase, but still have a secure password
  • In addition, you can add extensions on the end of you password to make it more secure. Some of examples of this include:
    • Gmail="gma"
    • Spotify="Spo"
  • Your password should be at least ten characters, but for better security you should have 12-15 characters
  • If you are a admin of a system, it is recommended that you change your password every thirty days, but if your are a user, then you should change your password every 90 days.
  • One password counts as one factor authentication, but for more security, use an additional method of security such as a biometric scanner


Setting password policies

  • If you are an admin, users on your network may not follow your password instructions, so you can follow these steps to control password policies


Enable Firewalls
  • There are no disadvantages in enabling a firewall, and it keeps you protected from attacks from the internet
    1. Open control panel by searching it in the app search in the bottom left corner
    2. Click into system and security and into Windows Firewall
    3. If it isn't on, click to turn it on on all networks
  • To let a process through the firewall or prevent processes from coming through the firewall
    1. Navigate back to the Windows Defender Firewall tab in control center
    2. Click "Allow an app or feature through windows firewall"
    3. To make changes, click "Change Settings"


Safe use of virtual machines

  • To use a virtual mackine safely, check the virtual machine you are about ot play's checksum against the one it is supposed to be to insure the VM is not corrupt
  • Never change the core files of a VM unless you are sure what you are doing because this will most likley corrupt the VM


Scan with windows defender

  • Windows Defender comes preinstalled on all windows machines. It scans your PC for viruses, without installing anything. To use it do these steps:
    1. Open settings by searching it in the bottom left corner
    2. Click into Update and Security
    3. Click on Windows Security
    4. Open Windows Defender Security Center
    5. Click on the shield and do a scan


Install windows updates

  • Windows Updates are crucial in ensuring that viruses dont get onto your PC. To use it do these steps:
    1. Open settings by searching it in the bottom left corner
    2. Click into Update and Security
    3. Search for new updates
    4. Download all updates
    5. Install all updates
    6. Restart your PC


File sharing permissions

  • To edit file permissions for specific users, do this:
    1. Open File Explorer
    2. Navigate to the file to select
    3. Right click your file
    4. Select properties
    5. Navigate to the security tab
    6. Click edit
    7. To add users, do these steps:
      1. Select add
      2. Type in the username
      3. Click Check Name
      4. Select permissions
      5. Click apply
    8. To remove users:
      1. Select remove
      2. Select apply
    9. To edit permissions:
      1. Select the user
      2. Check or uncheck permissions
      3. Click apply
